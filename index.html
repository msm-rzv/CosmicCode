Frequently Asked Questions

What prerequisites are needed for the beginner course?

Our beginner course, "Python Fundamentals: Getting Started," is designed for complete novices. No prior programming experience is required.

What support is available if I have questions or need help?

We offer email support for more specific questions.

What resources are included with the courses?

Each course includes in-person lessons, downloadable resources, coding exercises, and projects.

What software and tools do I need for the courses?

All you need is a computer with an internet connection. We recommend installing Python and a code editor like VSCode or PyCharm, which we will guide you through in the courses.

Are there any assessments or projects?

Yes, each course includes coding exercises, and projects to test your understanding and practical skills.

Do I need any specific hardware requirements?

Generally, any modern computer will suffice. Specific hardware requirements will be mentioned in the course if necessary.
